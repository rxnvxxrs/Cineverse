# 🎬 CineVerse

CineVerse is a curated movie discovery website built using HTML, CSS, and JavaScript.

## Features

- Modern dark-themed UI
- Movie search
- Genre browsing
- Curated collections
- Movie detail modal
- Reviews page
- About and Contact pages
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages
